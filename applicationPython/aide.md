# Pour l'application

## PyQt5 (interface graphique)
1. Vous allez avoir besoin de l'installer dans votre VSCode.
* En haut à gauche de votre fenêtre : ```Terminal > New Terminal```
* Dans cette fenêtre taper : 
    - (pour Windows) : ```pip install PyQt5``` 
    - ou (pour Linux) : ```sudo apt install python3-pyqt5```
* PyQt5 est maintenant installé et prêt à l'emploi ! 
2. C'est le moment de dire **Good Luck !**

## Cryptography (pseudonymisation)
1. Vous aurez aussi besoin de l'installer pour l'utiliser.
* Dans le même terminal (ou un différent peu importe) taper ; 
    - ```pip install cryptography```
    - ou ```sudo apt install python3-cryptography```
* Cryptography est lui aussi installé et prêt à l'emploi ! 
2. **Good Luck !**
* J'ai généré une clé que l'on utilisera pour les tests, je vous demande donc de **ne pas exécuter** le fichier ```secretKey.py``` afin que la clé reste la même.
* Merci d'avance !
